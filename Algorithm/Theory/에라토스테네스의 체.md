# 개요
~~~
- 고대 그리스의 수학자 에라토스테네스가 만들어 낸 소수를 찾는 방법
~~~
# 메커니즘
~~~
- 배열 생성
- 2부터 시작하여 자기자신을 제외한 특정 수의 배수에 해당하는 수를 순차적으로 제거
- 작업 수행 후 남아있는 수를 전체 출력
~~~
# 코드
~~~python
n=1000
a = [False,False] + [True]*(n-1)

primes=[]

for i in range(2,n+1):
  if a[i]:
    primes.append(i)
    for j in range(2*i, n+1, i):
        a[j] = False
print(primes)
~~~
