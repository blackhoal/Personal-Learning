# docker search
- 도커 레지스트리에서 제공하는 이미지를 검색하며 도커 CLI에서 실행  
~~~
docker search [options] term
~~~

# docker pull
- 도커 레지스트리에서 원하는 이미지를 로컬 머신에서 가져옴
~~~
docker pull image:tag
~~~
