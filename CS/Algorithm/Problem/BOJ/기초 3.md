# 11653 소인수 분해
https://www.acmicpc.net/problem/11653
## 노트
- 작성한 답안으로 실행하니 시간 초과 발생
~~~
[문제]
정수 N이 주어졌을 때, 소인수분해하는 프로그램을 작성하시오.

[Input]
72

[Output]
2
2
2
3
3
~~~
~~~python
# 틀린 답안
n = int(input())
prime_arr = []

for i in range(2, n) : 
    check = 0
    for j in range(2, i) :
        if i%j == 0 :
            check = 1
    if check != 1 :
        prime_arr.append(i)
        
for j in prime_arr : 
    while True:
        if n % j == 0 :
            n = int(n/j)
            print(j)
        else :
            break
    if n == 1 :
        break

# 타인 작성 답안
n = int(input())
i = 2

while n != 1 :
    if n % i == 0 :
        n = n / i
        print(i)
    else:
        i += 1
~~~
