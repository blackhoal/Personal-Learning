1. 배열의 필요성  
변수 선언의 편의성 확보  
배열을 이용하면 변수의 수가 무수히 많아도 하나의 문장으로 선언 가능  
순차적 접근
배열과 반복문을 사용할 경우 배열을 구성하는 변수에 순차적으로 접근하는 것이 가능  
~~~
// 일반적인 방식의 변수 선언 방식
num1 = num2 = num3 = 10;
num4 = num5 = num6 = 10;
num7 = num8 = num9 = 10;

// 배열 기반의 변수 선언 방식
for(int i=0;i<9;i++)
{
  numArr[i] = 10;
}
~~~
