# 빅데이터란?
정형 데이터 : 숫자나 문자로 쉽게 구분 가능한 데이터  
비정형 데이터 : 텍스트, 동영상, 음악 파일 등 구조화가 어려운 데이터  
|1세대 데이터|2세대 데이터|
|:------:|:---:|
|개인 컴퓨터|여러 디지털 기기|
|정형 데이터|비정형 데이터|
|기업 중심|개인 중심|
|예측 가능한 증가|예측 불가능한 증가|

## 빅데이터의 개념
|기관|정의|
|:------:|:---:|
|Gartner|인사이트와 의사결정을 위해 사용하며 체계적인 관리가 필요한 정보 자산|
|McKinsey|새로운 기술 체계에 따른 저장, 분석, 관리|
|IDC|가치를 추출할 수 있는 차세대 기술과 아키텍처|

## 빅데이터의 3V
1. 규모(Volume) : 정보의 범위 확대, 대량의 데이터
2. 다양성(Variety) : 새로운 종류의 데이터와 분석, 비전통적 형태의 미디어, 소셜 미디어 데이터
3. 속도(Velocity) : 실시간 정보, 신기술에서 유입되는 데이터

## 빅데이터를 통한 비즈니스 가치
1. 비즈니스 효율성
2. 마케팅 효과
3. 신규 비즈니스 창출

## 일반적인 빅데이터 접근 방식
~~~
요구사항 파악 → 인프라와 데이터 자원 및 분석 최적화
~~~

## 빅데이터 접근 방식
1. 양방향 중심 고객 분석  
2. 빅데이터 인프라  
> - 정보 통합
> - 확장 가능한 스토리지 인프라
> - 보안 및 관리
3. 빅데이터 도입 초기  
> - 내부 데이터와 신규 데이터를 적절히 활용한 분석 수행
> - ERP(Enterprise Resource Planning) : 기업 경영을 위한 영업, 회계, 생산 등의 시스템
> - BI(Business Intelligence) : 기업이 신속하고 정확한 비즈니스 의사결정을 위해 사용하는 데이터의 접근, 수집, 보관, 분석 등의 애플리케이션과 기술의 
4. 빅데이터 분석 역량  
~~~
정형 데이터 분석 → 반정형, 비정형 데이터 분석
~~~
> - 데이터 마이닝 : 대규모로 저장된 데이터 안에서 체계적이고 자동으로 통계적 규칙이나 패턴을 찾아내는 것
5. 빅데이터 도입 로드맵  
~~~
① 교육 : 빅데이터를 일부 이해한 기업이 왜 이것을 해야 하는지 연구하는 단계/ 지식 습득과 시장 관찰  
   ↓  
② 탐색 : 전사적 차원에서 공식적으로 진행되는 단계 / 요구사항과 과제를 바탕으로 전략과 계획 개발  
   ↓  
③ 참여 : 테스트 등을 통해 자사 빅데이터의 비즈니스 가치를 증명하고 도입 타당성을 알 수 있는 단계 / 빅데이터 시범 운영으로 요구사항의 타당성 검증  
   ↓  
④ 실행 : 더 넓은 범위에서 빅데이터를 운영하고 분석하는 단계빅데이터 활용 계획 및 지속적인 고급 분석 적용  
~~~
