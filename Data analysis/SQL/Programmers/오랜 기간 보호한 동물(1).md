# 풀이
~~~mysql
SELECT A.NAME, A.DATETIME 
FROM ANIMAL_INS A
LEFT JOIN ANIMAL_OUTS B
ON A.ANIMAL_ID = B.ANIMAL_ID
WHERE B.ANIMAL_ID IS NULL
ORDER BY DATETIME
LIMIT 3;
~~~
# 필기
~~~
- ANIMAL_OUTS에 없는 ANIMAL_INS의 ANIMAL_ID를 찾기 위해 WHERE B.ANIMAL_ID IS NULL 구문 사용
~~~
